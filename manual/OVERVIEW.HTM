Object Oriented Harbour GUI (ooHG) is a graphical interfase
for developing Microsoft Windows applications using (x)Harbour.
ooHG enables (x)Harbour programmers to easily develop applications
for Microsoft Windows with a graphical user interfase consistent
with Microsoft's "Common Controls".


To download, please visit
http://sourceforge.net/projects/oohg/files/

To ask for help, join the users group at
http://es.groups.yahoo.com/group/oohg/join

You can find more information at
http://www.oohg.org
